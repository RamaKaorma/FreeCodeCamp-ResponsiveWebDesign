Getting Started

C is a procedural language, one in which the computation is described in the order in which it is to be carried out.

The following are the strengths of C:

  • Programs are portable in C as it is universally available. There are a wide range of compilers on most computer architectures.
  • It is robust. It is a mature language controlled by formal standards.
  • It has various applications, and it is fast. In fact, the Unix system is written in C.
  • C is a close to machine language as there is a close connection between the constructs of the language and the machine it operates on. This is evident in C's code being able to directly work with memory.
